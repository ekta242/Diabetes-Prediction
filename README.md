# Diabetes Prediction

## About This Project

This project was completed as part of my machine learning internship. When I started the internship, I was excited to work on real machine learning projects and understand how predictive models are built from data.

The Diabetes Prediction project helped me understand the basics of machine learnin, data preprocessing, and model evaluation. More importantly, it gave me a practical understanding of some of the statistical concepts that are often discussed in theory.

I also noticed that I'm really inclined towards learning about the "Why" and "How" it works more than the coding.Exploring how the Statistical Reasoning behind the code felt more rewarding to me.
## What I Learned

While working on this project, I explored:

* Data cleaning and preprocessing
* IQR-based outlier detection and removal
* Handling class imbalance using SMOTE
* Feature selection and data preparation
* Logistic Regression for classification
* Model evaluation and prediction

One of my favorite parts of the project was understanding the reasoning behind these techniques. Learning how outliers affect data, why class imbalance matters, and how machine learning models make predictions helped the mathematical and statistical concepts make much more sense to me.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Imbalanced-learn (SMOTE)

## Project Workflow

1. Load and explore the dataset
2. Clean the data
3. Remove outliers using IQR
4. Balance classes using SMOTE
5. Split data into training and testing sets
6. Train a Logistic Regression model
7. Evaluate model performance

## Note

The dataset used in this project was provided as part of an internship/training program and is not included in this repository.
